JavaScript Programming Basics

JavaScript is the programming language of the web. Once you learn JavaScript you can start building interactive websites, apps, and games that run in the browser, on your phone, on your computer, or even on your robot!

In 10 weeks, you’ll learn the basics of computer programming: how to store data, use logic, and write functions. Each week, you will practice your coding skills on live webapps with the goal of building your very own chatbot.
